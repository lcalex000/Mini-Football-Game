# Baller League — Football Game Demo

A single-file HTML5 canvas football game, built as an exclusive partnership pitch to Baller League. Fast-paced 6-a-side gameplay, runs instantly in any browser on mobile or desktop, no download required.

## About

Me and Batch Binary has built a 6-a-side football game engine that runs instantly in any browser, on any device — completely free and with no download required. It can be shared anywhere: a message, an email, a blog, a YouTube description.

This demo is a pitch for one exclusive partner to bring it to life with full league and custom branding. Placeholder graphics only — all visuals are designed to be replaced to fit partner branding.

## Features

- Playable demo on mobile and desktop
- Fast-paced 6v6 gameplay
- Adjustable difficulty for all ages
- Fully child-safe
- Zero download, instant play
- Built for embed and sharing

## Roadmap (with exclusive partnership)

- Player collection and boost mechanics
- Multiplayer development
